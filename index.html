What I'm currently working on:

Playholdr

React ExpressJs MongoDB SocketIO

A multiplayer games site that features a versus Sudoku (more games coming soon) and focuses on making the time to start playing as fast as possible.

  • REST API to authenticate both guest and registered users
  • Sockets to update games and chats in real-time
  • MongoDB to store and retrieve user information, tokens, and sudoku boards
  • React to display and update game information in real-time
  • Jest to run automated tests for the front and backend
  • Github Actions to build and deploy Docker images to both a raspberry pi, and to production on AWS
